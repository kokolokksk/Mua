# catcatDM

> catcatDm! The meta-electron-catcat-bilibili-danmu application!

### 说明
喵喵酱牌 哔哩哔哩 直播弹幕姬（没有直播功能 只能看弹幕！）

- 发送弹幕需要设置 SESSDATA(SESSDATA) 与 csrf(bili_jct)，登陆哔哩哔哩后 F12 复制cookies中的 SESSDATA 与 bili_jct 对应键值
